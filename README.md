# 酒店管理系统+SSM

## 前言

欢迎来到酒店管理系统+SSM的项目介绍。本项目致力于为酒店行业提供一套全面、高效、易用的管理系统。基于Java语言和SSM框架（Spring、Spring MVC、MyBatis），结合微信小程序、前端技术（JS、Vue、CSS3、Uniapp），打造了一套功能完善、界面友好的酒店管理系统。

## 内容介绍

酒店管理系统主要包括以下功能模块：客房管理、预订管理、客户管理、财务管理、员工管理等。系统采用模块化设计，各模块之间相互独立，便于维护和扩展。此外，还提供了微信小程序端，方便客户在线预订、查询订单等。

## 技术介绍

- **语言**：Java
- **使用框架**：Spring、Spring MVC、MyBatis，微信小程序
- **前端技术**：JS、Vue、CSS3，Uniapp
- **开发工具**：IDEA/Eclipse，Uniapp
- **数据库**：MySQL 5.7/8.0
- **数据库管理工具**：phpstudy/Navicat
- **JDK版本**：jdk1.8
- **Maven**：apache-maven 3.8.1-bin
- **前端环境**：Node.Js 12\14\16
